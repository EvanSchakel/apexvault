## 2026-05-14 - Empty State Actionability
**Learning:** In CLI applications, displaying just a table header or a static "None" message when a dataset is empty fails to guide the user on how to fix the state. Empty states should provide direct action paths using the numbered menu choices (e.g., "Use option X to...").
**Action:** Always include a specific numeric call-to-action in empty states pointing to the menu item that populates the data.
