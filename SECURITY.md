# Security Policy

## Supported Versions

We only provide security updates for the latest major version.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

Security is highly important to us, especially given that ApexVault is a personal finance manager.

If you have discovered a security vulnerability, please report it via private correspondence. **Do not create a public issue**. Instead, please send an email to `security@apexvault.example.com` (placeholder) with the details of the vulnerability, including steps to reproduce.

We will endeavor to respond to your report within 48 hours and work with you to understand and resolve the issue safely before disclosing it publicly.
